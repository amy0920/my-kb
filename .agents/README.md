# Agent 規則索引

這個資料夾是所有 Agent 規則與交接模板的入口。
如果你剛接手這個 repo，先從共用規則開始，再依任務角色進入對應文件。

## 快速入口

| 項目 | 用途 |
| --- | --- |
| [共用規則](../AGENTS.md) | 所有 Agent 都必須遵守 |
| [使用手冊](./usage.md) | 了解工作流程、交接標準與常見錯誤 |
| [交接模板](./handoff.md) | 結束任務前填寫 |

## 角色文件

| 角色 | 文件 |
| --- | --- |
| Planner | [planner.md](./planner.md) |
| Research | [research.md](./research.md) |
| Implementation | [implementation.md](./implementation.md) |
| Review | [review.md](./review.md) |

## 閱讀順序

1. 先讀 [共用規則](../AGENTS.md)
2. 再讀 [使用手冊](./usage.md)
3. 讀自己的角色規則
4. 結束時填寫 [交接模板](./handoff.md)

## 角色切換

- 任務需要重新拆解時，交回 Planner。
- 任務需要補背景或查證時，交回 Research。
- 任務需要實際修改檔案時，交回 Implementation。
- 任務需要檢查品質與風險時，交回 Review。
